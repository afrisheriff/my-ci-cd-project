Project Overview
This project demonstrates a simple CI/CD pipeline using Github, Jenkin, Docker and Linux.

whenever code is pushed to Github: -Jenkins automatically pulls the latest code

Build a Docker image
Deploy the application inside a Docker container
Application becomes accessible in browser
Tools used
-Git -Github -Jenkins -Linux -Nginx

Pipeline Stages
Clone
Build Docker Image
Deploy Container
Project Architecture
Developer | Github Repository | Jenkins Pipeline | Docker Build | Docker Container Deployment | Browser Output

Jenkins Pipeline
-Clone source code from Github -Build Docker image using Dockerfile -Deploy container on port 8081
